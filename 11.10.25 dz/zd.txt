ДЗшечка 

1\. Напишите программу, которая принимает строку(не слово!) и 

проверяет, является ли она палиндромом, игнорируя 

пробелы, знаки препинания и регистр букв 

Пример: 

Вход: "А роза упала на лапу Азора" 

Выход: true 

2\. Напишите 

программу, 

которая 

принимает 

строку 

(предложение), разбивает ее на слова и возвращает первое 

слово, в котором содержится наибольшее количество гласных 

букв 

Пример: 

Вход: "Быстрая рыжая собака перепрыгнула через ленивого 

1. # Аскара" 

Выход: перепрыгнула 

3\. Напишите программу, которая  получает на вход двумерный 

массив слов и проверяет, что в каждом столбце есть хотя 

бы одна строка, оканчивающаяся на гласную 

Пример: 

Вход:  

{ 

{"кошка", "стол"}, 

{"собака", "стул"} 

} 

Выход: true 

4\. Напишите программу, которая выводит все строки длины N 

Пример: 

Вход: N = 2 

Выход: аа, ае, аё, аи, ао, ау, ... , яя (100 комбинаций) 

5\. Напишите программу, которая переворачивает массив (то 

есть строки делаем столбцами и наоборот) 

Пример: 

Вход:  

{ 

{"один", "два", "три"}, 

{"четыре", "пять", "шесть"} 

} 

Выход: 

{ 

{"один", "четыре"}, 

{"два", "пять"}, 

{"три", "шесть"} 

}
